Environment Setup

To ensure smooth execution, please set up the environment using Python 3.13.6 and run the following command to install all required dependencies:

pip install -r requirements.txt

The code installs:


pandas for data manipulation and analysis


numpy for numerical computations




Running the Notebook and Reproducing Results

Download the notebook (XXX.ipynb) and navigate to the directory where it is saved. Open the file using Jupyter Notebook.

Alternatively, if you are using Google Colab, upload the .ipynb file directly to Colab.

To execute the notebook, run the cells sequentially by clicking “Run” on each cell.
